SkillSense – Resume Analyzer

SkillSense  is a web-based resume analysis system that helps users understand their skills and suitable career paths.

-It analyzes resumes to:

-Extract important skills

-Identify the relevant domain

-Suggest suitable job roles

The system uses APIs and stores user data using PostgreSQL for better analysis and management.

 Features

-Resume skill extraction

-Domain detection

-Role suggestions

-API-based analysis

-Data storage using PostgreSQL

 Tech Stack

-React (Frontend)

-Node.js & Express (Backend)

-PostgreSQL (Database)

-REST APIs

Purpose

The main aim of this project is to help students and job seekers understand their resume strength and guide them towards suitable career roles.
